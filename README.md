# RV_Additions
This contains files that can be downloaded and printed to make a Recreational Vehicle more user friendly.

RemoteButtonPusher is a mechanism that provides a remote mechanical button to turn on a power station located in a hard to reach spot.

Roof_Insulaton provides clips to hold up half inch thick roof insulaton.

TentFlapVent provides spacers to hold out the bottom of a tent flap on a TopoTopper Mesa.

TentWallFan provides parts to mount a small fan in a tent wall.

WaterPump provides parts and an Ardino program to implement a water tank pump and display of water remaining.

TailGateLock provides a slider inside the tailgate to move the lock mechanism. 2023 Tacoma.

Eggs egg carton for three eggs.

Misc
ChannelWall clips to keep small parts from sliding in the upstairs side channels. (Mesa)
HitchWasher goes in a 2 inch hitch reciever.
TailGateWedge '23 Tacoma wedge to make the tailgate seal out dust better (need two)
WindowKnob provides a knob for the window to be glued on inside the topper.
